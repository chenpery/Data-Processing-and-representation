# 236201 - Data Processing and Representation
This repository contains all homework notebooks and code for the course **"Introduction to Data Processing and Representation" (236201)**.
Each homework explores a different concept in signal processing using both theory and implementation in Python.

## Contents

| File / Notebook                             | Description |
|--------------------------------------------|-------------|
| `solving the Lp problem using the L2 solution.ipynb` | Implements iterative approximation of Lp minimization using L2 solvers|
| `quantization.ipynb`                        | Implements uniform and Max-Lloyd quantization |
| `subsampling and reconstruction.ipynb`      | Explores subsampling an image and reconstructing it using various metrics|
